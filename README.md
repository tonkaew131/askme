# AskMe

Anonymously ask question, send songs / images / drawing to specific instagram account

## Getting Started

### Prerequisites

Install required package

- npm

```sh
npm install
```

### Installation

- Build production website

```sh
npm run build
```

- Run production server

```sh
npm start
```

## Usage

*https://[Domain]/ask/[AccountID]*
